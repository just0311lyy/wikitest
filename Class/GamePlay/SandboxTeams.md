# SandboxTeams
------------------------------------------------------------------------------------------
## 描述

此类是一个服务！它是顶级单例，可以使用`GetService`函数获取。
`Teams`服务用来存储游戏的`Team`对象，这些`Team`对象必须被设置为`Teams`服务的子项。

继承：`ServiceNode`
