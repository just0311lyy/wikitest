# Block
------------------------------------------------------------------------------------------
## 描述

方块数据；负责存储方块的`ID`和`blockdata`

------------------------------------------------------------------------------------------
## 属性：

|<div style="width:1125px">[int]() &emsp;[<font color="dd00dd">id</font>]()</div>|
|:---|
|方块的ID|

|<div style="width:1125px">[int]() &emsp;[<font color="dd00dd">data</font>]()</div>|
|:---|
|方块的数据（blockdata）例如楼梯通修改blockdata来改变楼梯的朝向|
